# 🚀 Rover Survival

> A terminal-based survival game built using **C++ Object-Oriented Programming**  
> Developed for **CSE 1254 - OOP Lab, RUET**

---

## 🎮 Gameplay Overview

- 🎯 Control the rover using **W, A, S, D**
- ⚡ Collect `*` energy to survive
- 🔫 Shoot enemies using **SPACE**
- 🪨 Hide near rocks `#` during storms
- ⏳ Survive as long as possible

---

## ⚙️ Features

- 🧠 Object-Oriented Design (Rover, Enemy, Grid, Laser)
- 🤖 Enemy AI (chasing + shooting)
- 🌪️ Dynamic storm survival mechanic
- ⚡ Energy system with regeneration
- ⏱️ Real-time survival timer
- 🖥️ ASCII-based rendering engine
- 🔲 Fullscreen console experience

---

## 🎯 Gameplay Goal

Survive as long as possible by:
- Managing energy ⚡
- Avoiding enemies 🤖
- Taking cover during storms 🌪️

---

## 🖥️ Controls

| Key     | Action        |
|---------|--------------|
| W       | Move Up      |
| S       | Move Down    |
| A       | Move Left    |
| D       | Move Right   |
| SPACE   | Shoot Laser  |
| Q       | Quit Game    |
